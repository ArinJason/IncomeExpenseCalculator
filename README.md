Income Expense Calculator A simple, responsive web app to track income and expenses with full CRUD, filters, totals, and localStorage persistence. ## Features - Add income/expense entries with description and amount - Edit and delete entries inline - Filters: all, income, expense (radio buttons) - Summary totals: total income, total expenses, net balance - Reset button to clear form inputs - Data persisted in localStorage across sessions - Clean, responsive UI for desktop and mobile - Vanilla HTML, CSS, and JavaScript (no frameworks) ## Tech stack - HTML - CSS - JavaScript
